# **App Name**: Consultant Asset Manager

## Core Features:

- Landing Dashboard: Visual dashboard providing high-level metrics such as Total Assets, Assets in Use, Open Tickets, Pending Requests, and Recent Activity using dummy data.
- Asset Management Section: Displays a structured asset table with Asset Tag, Asset Name, Type, Category, Location, Assigned User, Status, and Actions.
- Asset Detail Page: Shows asset information sections such as Overview, Assignment History, Lifecycle Status, and QR Code Preview, displaying a sample QR code and lifecycle timeline.
- Helpdesk / Ticketing Module: A table showing Ticket ID, Title, Priority, Status, Requester, Assigned Support, and Created Date. Supports the creation of new tickets with visual form elements.
- Reports Section: Visually represents insights with charts for asset utilization, ticket aging, ticket volume, and asset status distribution using dummy data.
- Audit Logs Page: Lists system actions in a chronological table format (e.g., Asset Created, Ticket Updated) with static data to demonstrate governance and compliance.
- Settings Section: Shows organization-level configuration screens such as Departments, Locations, Asset Categories, and User Roles, with tables and modal designs for visual completeness.

## Style Guidelines:

- Primary color: Muted Blue (#6495ED), reflecting calmness and enterprise trust.
- Background color: Off-white (#F5F5F5) providing a clean, minimal feel.
- Accent color: Greyish-Blue (#778899), creating depth and subtle highlighting.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, objective, neutral look that works well for enterprise dashboards. 
- Use professional, outline-style icons that align with enterprise UX best practices.
- Employ a structured layout with clear spacing, subtle shadows, consistent button styles, and breadcrumb navigation.
- Use subtle transitions and animations to enhance user experience without being distracting.
